           3，克隆项目
           4，本地操作添加代码并add，commit
           5，提交
                git fetch origin 从线上拉去最新代码
                切换到当前分支上 自己的分支
                然后合并最新dev代码到当前分支：git rebase origin/dev

                git push origin suyuhao

                组长：git fetch oriigin 从线上摘取最新所有分支代码
                通过 git merge 合并其他组员的分支到dev和master上
                最后
                git push origin master
                git push origin dev
                保持master和dev永远是最新代码
